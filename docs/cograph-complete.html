Requires R ≥ 4.1.0. Core imports: R6, grid, grDevices, ggplot2. Optional: igraph, network, qgraph, tna, scales, RColorBrewer, colorspace, gifski.

Core Functions

cograph() plotonly

The main entry point. Accepts adjacency matrices, edge lists, igraph, statnet network, qgraph, or tna objects and creates a visualization-ready network object.

cograph(input, layout = "spring", directed = NULL, node_labels = NULL, seed = 42, ...)
ParameterDescriptionDefault
inputSquare matrix, edge list data frame, igraph, statnet, qgraph, or tna object—
layout"spring", "circle", "groups", "grid", "random", "star", "bipartite", "oval""spring"
directedForce directed interpretation. NULL for auto-detectNULL
node_labelsCharacter vector of node labelsNULL
seedRandom seed for deterministic layouts42
Returns: A cograph_network object that can be further customized and rendered.
Example 
# From adjacency matrix
adj <- matrix(c(0, 1, 1, 1, 0, 1, 1, 1, 0), nrow = 3)
cograph(adj)

# Pipe-friendly workflow
cograph(adj, layout = "circle") |>
  sn_nodes(fill = "steelblue") |>
  sn_edges(color = "gray50") |>
  splot()

as_cograph() plotonly

Creates a lightweight cograph_network object from various network inputs. The resulting object is a named list with all data accessible via $.

as_cograph(x, directed = NULL, ...)
ParameterDescriptionDefault
xSquare matrix, edge list, igraph, statnet, qgraph, tna, or cograph_network object—
directedForce directed interpretation. NULL for auto-detectNULL
Returns: A cograph_network named list with $nodes, $edges, $weights, $directed, $source.

from_qgraph() plotonly

Convert a qgraph object to cograph parameters. Extracts the network, layout, and all relevant arguments from a qgraph object.

from_qgraph(qgraph_object, engine = c("splot", "soplot"), plot = TRUE, weight_digits = 2, show_zero_edges = FALSE, ...)
Returns: Invisibly, a named list of cograph parameters passable to splot() or soplot().

from_tna() plotonly

Convert a tna object to cograph parameters. Extracts the transition matrix, labels, and initial state probabilities.

from_tna(tna_object, engine = c("splot", "soplot"), plot = TRUE, weight_digits = 2, show_zero_edges = FALSE, ...)

Plotting Engines

splot() plotonly

The primary plotting function using base R graphics. Fast, flexible, and feature-rich.

splot(x, layout = "spring", directed = NULL, threshold = 0, labels = NULL, node_size = 5, node_fill = NULL, node_shape = "circle", node_border_color = "#333333", edge_color = NULL, edge_width = NULL, edge_labels = FALSE, edge_label_size = 0.7, curvature = NULL, arrow_size = 0.4, title = NULL, theme = "default", ...)
Key parameters
ParameterDescriptionDefault
xNetwork input (matrix, igraph, tna, cograph_network)—
layoutLayout algorithm or coordinate matrix"spring"
node_sizeNode size (numeric or vector)5
node_fillNode fill color(s)NULL
edge_colorEdge color or "weight" for autoNULL
edge_labelsShow edge labelsFALSE
curvatureEdge curvature (scalar or vector)NULL
donut_fillPer-node donut proportionsNULL
Example 
# Basic plot
adj <- matrix(c(0, .5, .3, .5, 0, .4, .3, .4, 0), 3, 3)
rownames(adj) <- colnames(adj) <- c("A", "B", "C")
splot(adj)

# With customization
splot(adj, layout = "circle",
      node_fill = c("red", "blue", "green"),
      node_size = c(6, 8, 10),
      edge_color = "weight",
      edge_labels = TRUE)

Network Analysis

centrality() merged

Computes centrality measures for nodes in a network. Returns a tidy data frame with one column per measure.

centrality(x, measures = "all", mode = "all", normalized = FALSE, weighted = TRUE, directed = NULL, loops = TRUE, simplify = "sum", digits = NULL, sort_by = NULL, ...)
ParameterDescriptionDefault
xNetwork input (matrix, igraph, cograph_network, tna)—
measures"all" or vector: "degree", "strength", "betweenness", "closeness", "eigenvector", "pagerank", "authority", "hub", "eccentricity", "coreness", "constraint", "transitivity""all"
modeFor directed: "all", "in", "out""all"
normalizedNormalize to 0-1 rangeFALSE
sort_byColumn name to sort results by (descending)NULL
Returns: Data frame with columns: node, plus one column per measure (with mode suffix for directional measures).
Example 
adj <- matrix(c(0, 1, 1, 1, 0, 1, 1, 1, 0), 3, 3)
rownames(adj) <- colnames(adj) <- c("A", "B", "C")

# All centrality measures
centrality(adj)

# Specific measures, sorted
centrality(adj, measures = c("degree", "betweenness", "pagerank"),
           sort_by = "pagerank", digits = 3)

# Directed network
centrality(adj, directed = TRUE, mode = "in")
